# card UI
This is a simple Card UI project made using HTML and CSS. I used Flexbox to center the card on the page and align its content properly. The card has a soft background color with a backdrop-filter(blur effect) to give it a frosted glass look. I also added a border and box-shadow to make the card look smooth and slightly raised from the background. This project helped me practice basic css styling techniques like layput, colors, shadows, and blur effects, and gave me a better understanding of how small design details can make a UI clean and modern.
## Tech Used
-HTML
-CSS(Flexbox,backdrop-filter,box-shadow,border and padding)
